- GitHub repo е»әиӯ°е…Ҳз”ұдҪҝз”ЁиҖ…еңЁз¶Ій Ғдәәе·Ҙе»әз«ӢпјҢзўәиӘҚеҗҚзЁұгҖҒprivate/public иҲҮж¬ҠйҷҗеҫҢпјҢеҶҚз”Ё Git жҢҮд»ӨдёҠеӮігҖӮ
- GitHub CLI дёҚеҲ—е…Ҙй җиЁӯжөҒзЁӢпјҢеҸӘдҪңзӮәжңӘдҫҶйңҖиҰҒиҮӘеӢ•еҢ–жҷӮзҡ„еҸҜйҒёж–№ејҸгҖӮ
- еҹ·иЎҢ `git push` жҲ–иЁӯе®ҡ / дҝ®ж”№ remote еүҚпјҢеҝ…й ҲзўәиӘҚжң¬ж©ҹе°ҲжЎҲеҗҚзЁұ / и·Ҝеҫ‘гҖҒREADME жҲ–е°ҲжЎҲиӘӘжҳҺгҖҒbranchгҖҒupstream иҲҮ remote repo еҗҚзЁұжҲ–жә–еӮҷиЁӯе®ҡзҡ„ repo URL иғҪеҗҲзҗҶе°ҚжҮүеҲ°еҗҢдёҖеҖӢе°ҲжЎҲпјӣиӢҘзңӢиө·дҫҶеғҸдёҚеҗҢе°ҲжЎҲжҲ–иҲҮдҪҝз”ЁиҖ…жҳҺзўәжҢҮе®ҡзӣ®жЁҷдёҚз¬ҰпјҢеҝ…й Ҳе…ҲйҖҡзҹҘдҪҝз”ЁиҖ…дёҰзӯүеҫ…еҶҚж¬ЎзўәиӘҚгҖӮ
- иӢҘж—ўжңү remote зңӢиө·дҫҶжҢҮеҗ‘дёҚеҗҢе°ҲжЎҲпјҢдёҚеҫ—иҮӘиЎҢиҰҶи“ӢжҲ– pushпјҢеҝ…й Ҳе…Ҳеӣһе ұе·®з•°гҖӮ

## жӯ·зЁӢйҮҚй»һ

- иЁҺи«–еҫҢжұәе®ҡжҺЎз”ЁзІҫз°Ўзөҗж§ӢпјҢйҒҝе…ҚдёҖй–Ӣе§ӢеҠ е…ҘйҒҺеӨҡз®ЎзҗҶж–Үд»¶гҖӮ
- иЁҺи«–еҫҢжұәе®ҡдҪҝз”Ё `AGENT_PROGRESS.md` иЁҳйҢ„зӣ®еүҚйҖІеәҰгҖҒйҮҚй»һиЁҺи«–гҖҒеҫ…зўәиӘҚдәӢй …иҲҮдёӢдёҖжӯҘгҖӮ
- иЁҺи«–еҫҢжұәе®ҡиіҮж–ҷеӨҫиӘӘжҳҺеҜ«еңЁ `AGENTS.md` иҲҮ `README.md`пјҢдёҚеңЁжҜҸеҖӢиіҮж–ҷеӨҫеҗ„ж”ҫиӘӘжҳҺжӘ”гҖӮ
- иЁҺи«–еҫҢжұәе®ҡдҝ®ж”№ `.env` жҷӮеҝ…й ҲеҗҢжӯҘжӣҙж–° `.env.example`гҖӮ
- еҸғиҖғ `poly-rewards4` иҰҸеүҮеҫҢпјҢжұәе®ҡиЈңеј· `.env.example` иӘӘжҳҺдҫҶжәҗгҖҒйқһж•Ҹж„ҹй җиЁӯеҖјдёҖиҮҙиҲҮд»»еӢҷж”¶е°ҫжӘўжҹҘиҰҸеүҮгҖӮ
- иЁҺи«–еҫҢжұәе®ҡеҠ е…Ҙ `.gitkeep`пјҢйҒҝе…Қз©әиіҮж–ҷеӨҫеңЁ Git дёӯж¶ҲеӨұгҖӮ
- иЁҺи«–еҫҢжұәе®ҡиЈңдёҠ Git дёҠеӮіжөҒзЁӢпјҢжҺЎз”Ёдәәе·Ҙе»әз«Ӣ GitHub repo еҠ жң¬ж©ҹ Git жҢҮд»Ө push зҡ„дҝқе®ҲжөҒзЁӢгҖӮ
- иЁҺи«–еҫҢжұәе®ҡй–Ӣе§Ӣе°Үзӣ®еүҚе°ҲжЎҲдёҠеӮіеҲ° Gitпјӣе·Іе®ҢжҲҗжң¬ж©ҹ Git еҲқе§ӢеҢ–гҖҒе®үе…ЁжӘўжҹҘиҲҮз¬¬дёҖеҖӢ commitпјҢдёҰиЁӯе®ҡ remote зӮә `https://github.com/billshiou2/00_project-template.git`гҖӮ
- еҳ—и©Ұ push жҷӮпјҢз¶Іи·ҜжІҷзӣ’е…Ҳйҳ»ж“ӢйҖЈз·ҡпјӣдҪҝз”ЁиҖ…жү№еҮҶеҫҢпјҢе·ІжҲҗеҠҹжҺЁйҖҒеҲ° GitHubгҖӮ
- еӣ  Git еҒөжё¬ repo ownership иҲҮзӣ®еүҚ Windows дҪҝз”ЁиҖ…дёҚеҗҢпјҢе·Ідҫқ Git жҸҗзӨәе°Ү `C:/# code/# project-template3` еҠ е…Ҙе…Ёеҹҹ `safe.directory`гҖӮ
- дҪҝз”ЁиҖ…жҸҗйҶ’жӣҫзҷјз”ҹ Git дёҠеӮійҢҜ repo зҡ„йўЁйҡӘпјҢеӣ жӯӨе·Іе°Ү push еүҚе°ҲжЎҲеҗҚзЁұ / remote / branch ж ёе°ҚжөҒзЁӢеҜ«е…Ҙ `AGENTS.md`гҖҒ`docs/git-workflow.md` иҲҮ `README.md`пјӣеҫҢзәҢеҸҲзўәиӘҚдёҚиҰҒжұӮеҗҚзЁұйҖҗеӯ—зӣёеҗҢпјҢеҸӘиҰҒжұӮиғҪеҗҲзҗҶе°ҚжҮүгҖӮ
- дҪҝз”ЁиҖ…иҰҒжұӮ push жң¬ж¬Ў Git дёҠеӮіеүҚзўәиӘҚиҰҸеүҮжӣҙж–°пјӣpush еүҚе·Іж ёе°Қзӣ®еүҚи·Ҝеҫ‘гҖҒremoteгҖҒbranch иҲҮи®ҠжӣҙжӘ”жЎҲиғҪеҗҲзҗҶе°ҚжҮүеҲ°жң¬е°ҲжЎҲгҖӮ

## дёӢдёҖжӯҘ

- дҫқеҜҰйҡӣе°ҲжЎҲйңҖжұӮж–°еўһеҺҹе§ӢзўјгҖҒиЁӯе®ҡжҲ–ж–Үд»¶гҖӮ
- иӢҘж–°еўһеҘ—д»¶пјҢеҝ…й ҲйҒөе®Ҳ `AGENTS.md` зҡ„еҘ—д»¶зүҲжң¬е®үе…ЁиҰҸеүҮгҖӮ
- ж–°е°ҲжЎҲз¬¬дёҖж¬ЎдёҠеӮі GitHub жҷӮпјҢеҸғиҖғ `docs/git-workflow.md`гҖӮ
- зӣ®еүҚ `main` е·ІиЁӯе®ҡиҝҪи№Ө `origin/main`пјҢеҫҢзәҢдёҖиҲ¬жӣҙж–°еҸҜдҪҝз”Ё `git push`гҖӮ
